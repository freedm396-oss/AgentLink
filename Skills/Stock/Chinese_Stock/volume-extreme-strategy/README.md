# 成交量地量见底策略 (Volume Extreme Bottom Strategy)

基于成交量极度萎缩识别股价底部区域，捕捉地量见底后的反弹机会。

## 📊 策略概述

地量见底是A股市场经典的反转信号策略，适用于下跌趋势尾声的股票筛选。

### 核心逻辑

```
买入条件：
1. 成交量极度萎缩（< 20日均量的 40%）
2. 股价处于相对低位（较近期高点跌幅 > 20%）
3. 股价位于重要均线上方（MA20）
4. 出现地量后价格企稳（K线形态收窄）
5. 随后出现温和放量上涨

卖出规则：
- 止损：跌破地量当日最低价 -3% 或 固定 -8%
- 止盈：前高附近 或 涨幅 15-20%
- 时间止损：买入后 10 日无明显反弹则离场
```

### 策略表现

| 市场环境 | 胜率 | 盈亏比 | 持仓周期 | 适用场景 |
|---------|------|--------|---------|---------|
| 下跌末期 | 65-75% | 2.5:1 | 5-15天 | 底部反弹 |
| 震荡筑底 | 55-65% | 2:1 | 7-15天 | 波段操作 |
| 下跌中继 | < 50% | 不建议 | - | 空仓观望 |

## 📖 策略原理

### 1. 地量的意义

- **极度缩量**：表明市场情绪极度低迷，抛压枯竭
- **价格企稳**：地量之后价格不再创新低
- **均线支撑**：股价位于重要均线（MA20）上方

### 2. 买入信号确认

| 条件 | 说明 | 权重 |
|------|------|------|
| 地量程度 | 成交量 < 20日均量 40% | 35% |
| 价格位置 | 位于 MA20 上方 | 25% |
| 企稳信号 | 地量后 K 线收窄不创新低 | 25% |
| 后续确认 | 温和放量上涨 | 15% |

### 3. 风险控制

- **仓位管理**：单票不超过 20%，总仓位不超过 60%
- **止损设置**：-5% 至 -8% 机械止损
- **分批建仓**：首次 50%，确认后 50%

## ⚙️ 配置说明

### 策略参数 (config/strategy_config.yaml)

```yaml
# 成交量参数
volume_ma_period: 20        # 成交量均线周期
extreme_ratio: 0.4          # 地量阈值（均量的40%）
significant_ratio: 0.5      # 显著缩量阈值

# 价格参数
ma_period: 20              # 价格均线周期
min_deviation: 5           # 最低偏离幅度（%）
```

### 评分权重 (config/scoring_weights.yaml)

```yaml
weights:
  volume_extreme: 0.35       # 地量程度 35%
  price_position: 0.25      # 价格位置 25%
  stability_signal: 0.25     # 企稳信号 25%
  follow_up: 0.15            # 后续确认 15%
```
