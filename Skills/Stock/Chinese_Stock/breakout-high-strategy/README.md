# 突破前期高点策略 (Breakout High Strategy)

基于股价突破前期重要高点的技术分析策略，捕捉强势股突破后的主升浪机会。

## 📊 策略概述

突破前期高点是A股市场经典的追涨策略，适用于识别股价突破关键阻力位后的加速上涨行情。

### 核心逻辑

```
买入条件：
1. 股价突破前60日最高价（突破阈值2%）
2. 突破时成交量明显放大（>前20日均量1.5倍）
3. 短期动量为正（5日涨幅>0）
4. 突破力度与量能配合良好

卖出规则：
- 止损：跌破突破当日最低价 或 固定-7%
- 止盈：涨幅20-30%后分批止盈
- 时间止损：买入后3日无表现则离场
- 跌破前高：重新跌回突破价位下方
```

### 策略表现

| 市场环境 | 胜率 | 盈亏比 | 持仓周期 | 适用场景 |
|---------|------|--------|---------|---------|
| 上升趋势 | 68-75% | 3:1 | 5-15天 | 主升浪启动 |
| 震荡突破 | 55-65% | 2:1 | 3-10天 | 箱体突破 |
| 弱势突破 | < 50% | 1.5:1 | 2-5天 | 假突破风险高 |

## 📖 策略原理

### 1. 突破前期高点的意义

- **阻力变支撑**：前期高点是重要的阻力位，突破后转为支撑
- **筹码解放**：解放套牢盘，显示资金实力
- **趋势确认**：突破确认上升趋势的延续或反转

### 2. 买入信号确认

| 条件 | 说明 | 权重 |
|------|------|------|
| 突破强度 | 突破幅度 >2%，越大越好 | 30% |
| 量能确认 | 成交量 >前20日均量1.5倍 | 25% |
| 价格动量 | 5日涨幅 >0，短期趋势向上 | 20% |
| 形态质量 | 突破幅度与量能配合度 | 15% |
| 市场环境 | 趋势强度+量能配合+突破有效性 | 10% |

### 3. 风险控制

- **仓位管理**：单票不超过15%，总仓位不超过50%
- **止损设置**：-5%至-7%机械止损
- **分批建仓**：首次60%，确认后40%

## ⚙️ 配置说明

### 策略参数 (config/strategy_config.yaml)

```yaml
# 突破参数
breakout_settings:
  lookback_days: 60           # 回看60日最高价
  breakout_threshold: 0.02    # 突破确认阈值2%
  volume_multiplier: 1.5      # 成交量放大倍数

# 动量参数
momentum_settings:
  lookback_days: 5            # 5日动量
  min_momentum: 0             # 最小动量要求

# 买入确认
entry_confirmation:
  min_breakout_ratio: 0.02    # 最小突破幅度2%
  min_volume_ratio: 1.2       # 最小量比1.2倍
```

### 评分权重 (config/scoring_weights.yaml)

```yaml
weights:
  breakout_strength: 0.30     # 突破强度 30%
  volume_confirm: 0.25        # 量能确认 25%
  price_momentum: 0.20        # 价格动量 20%
  pattern_quality: 0.15       # 形态质量 15%
  market_environment: 0.10    # 市场环境 10%
```

## 🎯 评分细则

### 突破强度得分 (0-30分)

| 突破幅度 | 得分 | 说明 |
|---------|------|------|
| >8% | 30分 | 突破力度极强 |
| >5% | 25分 | 突破力度较强 |
| >3% | 20分 | 突破力度良好 |
| >2% | 15分 | 突破力度一般 |

### 量能确认得分 (0-25分)

| 量比 | 得分 | 说明 |
|------|------|------|
| >2.0 | 25分 | 成交量大幅放大 |
| >1.5 | 20分 | 成交量明显放大 |
| >1.2 | 15分 | 成交量温和放大 |
| <1.2 | 8分 | 成交量未明显放大 |

### 市场环境得分 (0-10分)

基于三项指标综合评分：
- **趋势强度** (0-4分)：5日动量越大得分越高
- **量能配合** (0-3分)：量比越高得分越高
- **突破有效性** (0-3分)：突破幅度与量能配合度

## ⚠️ 风险提示

1. **假突破风险**：股价突破后快速回落，需严格止损
2. **追高被套**：突破后涨幅过大时不宜追高
3. **大盘影响**：弱势市场突破成功率降低
4. **量能萎缩**：突破后量能快速萎缩需警惕
