# 持仓实时监控 SKILL（v3.1）

> 每个交易日 9:35-14:55 每5分钟自动监控持仓，基于买入策略类型给出差异化止盈/止损建议。

---

## 核心定位

- **差异化策略**：不同 `buy_reason`（买入原因）对应不同止盈/止损参数
- **止盈优先**：盈利保护最重要，达到目标即触发止盈
- **三层过滤**：技术面 → 消息面 → 宏观确认
- **非预测型**：严格监控当下数据是否符合「操作标准」

---

## 持仓文件格式

持仓文件（默认 `Chinese_Stock/my_holdings/holdings.json`，脚本自动读取）：

```json
[
  {
    "code": "600105",
    "name": "永鼎股份",
    "buy_price": 38.10,
    "shares": 1000,
    "position_ratio": 0.20,
    "position_value": 38100,
    "buy_reason": "breakout_high",
    "buy_date": "2026-04-11"
  }
]
```

## buy_reason 策略分类

| buy_reason | 含义 | 策略类型 | 单只上限 | 止损线 |
|------------|------|---------|---------|--------|
| `limit_up` | 涨停板连板 | 激进（主升浪） | 20% | -7% |
| `breakout_high` | 突破新高 | 激进（主升浪） | 20% | -7% |
| `gap_fill` | 缺口填充 | 稳健 | 15% | -5% |
| `ma_bullish` | 均线多头 | 稳健 | 15% | -5% |
| `macd_divergence` | MACD底背离 | 保守 | 10% | -3% |
| `rsi_oversold` | RSI超卖 | 保守 | 10% | -3% |
| `morning_star` | 早晨之星 | 保守 | 10% | -3% |
| `volume_extreme` | 地量见底 | 保守 | 10% | -3% |
| `a_stock_1430` | 尾盘超短（不过夜） | 超短 | 5% | -2% |

## 大盘状态自动判断

系统根据上证、深证、创业板三大指数的实时涨跌自动判断大盘环境（无需手动配置）：

| 自动判断 | 条件 | 总仓位上限 |
|---------|------|-----------|
| `strong` | 三指数平均涨幅 >1.5%，至少2个上涨 | 80% |
| `bullish` | 三指数平均涨幅 >0.5%，至少2个上涨 | 60% |
| `neutral` | 三指数平均涨跌幅在 -1%~1% 之间 | 50% |
| `bearish` | 三指数平均跌幅 >1%，或至少2个跌幅>1.5% | 30% |
| `weak` | 三指数平均跌幅 >2%，且至少2个跌幅>1.5% | 10% |

> 如需手动覆盖，可通过 `--market-status` 参数或配置文件指定。

## 止盈配置

### 激进（limit_up / breakout_high）—— 让利润奔跑

| 盈利幅度 | 减仓比例 | 操作 |
|---------|---------|------|
| ≥10% | 0% | 继续持有 |
| ≥20% | 0% | 继续持有 |
| ≥30% | 30% | 减仓30%，留70%持仓博主升浪 |
| ≥40% | 50% | 减仓50% |
| ≥50% | 70% | 减仓70% |
| ≥80% | 100% | 高位清仓 |

### 稳健（ma_bullish / gap_fill）—— 正常止盈

| 盈利幅度 | 减仓比例 | 操作 |
|---------|---------|------|
| ≥5% | 0% | 继续持有 |
| ≥10% | 30% | 减仓30% |
| ≥15% | 50% | 减仓50% |
| ≥20% | 50% | 减仓50% |
| ≥30% | 70% | 减仓70%，留30% |
| ≥40% | 100% | 清仓 |

### 保守（macd_divergence / rsi_oversold / morning_star / volume_extreme）—— 尽早落袋

| 盈利幅度 | 减仓比例 | 操作 |
|---------|---------|------|
| ≥5% | 20% | 减仓20% |
| ≥8% | 30% | 减仓30% |
| ≥10% | 50% | 减仓50%，过半仓已落袋 |
| ≥15% | 70% | 减仓70% |
| ≥20% | 100% | 清仓 |

### 超短（a_stock_1430）—— 次日开盘必走

| 盈利幅度 | 减仓比例 | 操作 |
|---------|---------|------|
| ≥2% | 50% | 减半仓 |
| ≥5% | 100% | 清仓不过夜 |

## RSI 辅助判断（过买干预）

即使盈利未达目标阶段，RSI 过高时也会建议减仓：

| 策略类型 | RSI 阈值 | 触发条件 |
|---------|---------|---------|
| 激进 | >90 | RSI>90才干预 |
| 稳健 | >85 | RSI>85适当加码 |
| 保守 | >80 | RSI>80就要注意 |
| 超短 | >75 | 超短 RSI>75就走 |

---

## 三层过滤机制

### 第一层：技术面（每5分钟检查）

| 指标 | 触发条件 | 权重 |
|------|----------|------|
| RSI | >80 超买 / <30 超卖 | 35% |
| MACD | 高位死叉 / 顶背离 | 30% |
| 成交量 | 价涨量缩 / 高位巨量 | 20% |

### 第二层：消息面确认

- 实时检索过去1-2小时新闻
- 情绪打分：利好 / 中性 / 利空 / 利好出尽
- 突发利空 → 加重减仓

### 第三层：宏观确认

- 上证/深证/创业板是否系统性下跌
- 北向资金异动
- 系统性风险 → 全局减仓

| 策略类型 | 强制止损线 |
|---------|----------|

## 操作信号类型

| 信号 | 含义 |
|------|------|
| 💰 止盈建议 | 盈利达标，触发止盈 |
| 🚨 止损 | 触及硬性止损线，强制清仓 |
| 🔴 减仓警告 | RSI/MACD 偏空，建议减仓 |
| 🟢 加仓 | 超跌/趋势确认，可加仓摊薄或扩大收益 |
| 🟡 轻度注意 | 需密切关注，暂无操作 |
| 🟢 继续持有 | 安心持有 |

## 调仓分析功能

**触发方式：** 将其他策略推荐的股票写入推荐列表，运行调仓分析脚本

### 推荐列表格式（`~/.openclaw/stock/recommendations.json`）

```json
[
  {
    "code": "600105",
    "name": "永鼎股份",
    "source": "breakout_high",
    "recommend_date": "2026-04-12",
    "entry_price": 38.50,
    "target_reason": "突破新高，趋势确认"
  }
]
```

### 四维度打分模型

| 维度 | 权重 | 指标说明 |
|------|------|---------|
| 技术结构 | 30分 | RSI、MACD、均线多头/空头 |
| 市场热度 | 25分 | 换手率、涨幅、量比 |
| 市场情绪 | 20分 | 当日涨跌幅情绪近似 |
| 资金流向 | 25分 | 主力净流入、北向资金 |

**总分 100 分，得分差 ≥10 分触发调仓信号，≥20 分触发强烈建议**

### 调仓分析脚本用法

```bash
cd ~/.npm-global/lib/node_modules/openclaw/skills/Stock/Chinese_Stock/sell-monitor/scripts
python3 rebalance.py
```

| 推荐列表 `recommendations.json` | 14:30 / 15:30 |
